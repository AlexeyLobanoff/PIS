# Репозиторий для КП по ПИС

## Clone

```sh
git clone https://github.com/AlexeyLobanoff/PIS.git
cd PIS
```

## Prepare

1. ```sh
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    ```
2. Create `.env` file in root directory
3. Fill `.env` file with environment variables as in an example `.env.example`

## Run

```sh
python main.py
```
